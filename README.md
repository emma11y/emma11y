# Emmanuelle ABOAF

## Présentation

Sourde de naissance et bionique avec mes deux implants cochléaires, je suis développeuse fullstack et coach en accessibilité chez [Shodo](https://www.shodo.io). Dans mon monde idéal, tout doit être accessible aussi bien dans la vraie vie que sur le Web.

J'accompagne les équipes dans le développement de leurs projets en intégrant l'accessibilité numérique, les méthodologies Craft et le TDD. Parallèlement, j'explore et implémente les principes du Context Engineering pour garantir le respect des règles du RGAA.

Je suis également conférencière et [Leadeuse Yeeso Paris](https://yeeso.fr/).
